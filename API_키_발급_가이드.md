# 🔑 API 키 발급 완벽 가이드

업비트와 빗썸에서 API 키를 안전하게 발급받는 방법입니다.

---

## ⚠️ 시작하기 전에 꼭 읽으세요!

### 중요 보안 수칙

```
✅ 해야 할 것:
  - 자산 조회 권한만 부여
  - 주문 권한만 부여
  - API 키를 안전하게 보관
  - 2단계 인증 설정

❌ 절대 하지 말아야 할 것:
  - 출금 권한 부여 (절대!)
  - API 키를 다른 사람과 공유
  - API 키를 깃허브에 올리기
  - API 키를 스크린샷 찍어 SNS 공유
```

#### 안전하게 보관하는 방법

1. **메모장에 저장**
   ```
   [업비트 API]
   Access Key: ABC123...xyz
   Secret Key: DEF456...uvw
   발급일: 2024-02-05
   ```

2. **파일 저장**
   - 파일명: `upbit_api_keys.txt`
   - 위치: 안전한 로컬 폴더
   - **깃허브에 절대 업로드 금지!**

3. **백업**
   - USB 메모리에 복사
   - 또는 비밀번호 관리 앱 사용

## 🔒 보안 강화 팁

### 1. IP 화이트리스트 설정

**내 IP 주소 확인:**
- 🔗 [whatismyipaddress.com](https://whatismyipaddress.com)

**설정 방법:**
```
API 키 발급 시
┌─────────────────────────────────┐
│  특정 IP에서만 허용              │
│  ┌───────────────────┐          │
│  │ 123.456.789.012   │  ◀ 입력  │
│  └───────────────────┘          │
└─────────────────────────────────┘
```

**장점:**
- 다른 IP에서 접속 시 API 차단
- 해킹 시도 방어

**단점:**
- IP가 바뀌면 재설정 필요
- 여러 장소에서 사용 시 불편

---

### 2. 2단계 인증 (OTP) 설정

**필수!**

1. **Google Authenticator 설치**
   - iOS: App Store
   - Android: Play Store

2. **업비트/빗썸 설정**
   - 보안 설정 → OTP 등록
   - QR 코드 스캔

3. **로그인 시 OTP 입력**

---

### 3. API 키 정기 갱신

**권장 주기: 3개월**

1. 기존 API 키 삭제
2. 새 API 키 발급
3. 프로그램에 새 키 입력

### 방법 2: .env 파일 사용 (고급)

나중에 배포할 때 사용!

1. **.env 파일 생성**
   ```
   UPBIT_ACCESS_KEY=ABC123...xyz
   UPBIT_SECRET_KEY=DEF456...uvw
   
   BITHUMB_ACCESS_KEY=GHI789...rst
   BITHUMB_SECRET_KEY=JKL012...opq
   ```

2. **주의사항**
   - 이 파일은 깃허브에 올리면 안 됨!
   - `.gitignore`에 포함됨

## 🆘 문제 해결

### "API 키가 작동하지 않습니다"

**확인사항:**

1. **키 복사가 정확한지**
   - 공백 없이 복사
   - 전체가 복사되었는지

2. **권한 설정 확인**
   - 자산 조회: ✅
   - 주문하기: ✅

3. **IP 화이트리스트**
   - 설정했다면 현재 IP 확인

---

### "OTP를 잃어버렸습니다"

**해결방법:**

1. 고객센터 문의
2. 본인 인증 후 재설정
3. 새 API 키 발급

---

### "API 키를 깃허브에 올렸습니다!"

**긴급 조치:**

1. **즉시 API 키 삭제**
   - 업비트/빗썸 > API 관리 > 삭제

2. **자산 확인**
   - 이상 거래 확인

3. **새 키 발급**

4. **비밀번호 변경**

5. **2단계 인증 재설정**
