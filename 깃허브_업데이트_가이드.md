## 🚀 방법 1: 웹에서 직접 수정 (가장 쉬움!)

### Step 1: 깃허브 로그인

1. **브라우저에서 깃허브 접속**
   - 🔗 [github.com](https://github.com)
   
2. **로그인**

---

### Step 2: 본인 저장소 찾기

1. **우측 상단 프로필 사진 클릭**

2. **"Your repositories" 클릭**

3. **본인 프로젝트 저장소 클릭**
   - 예: `crypto-trading-bot`

---

### Step 3: app.py 파일 수정

#### 3-1. 파일 열기

```
[깃허브 저장소 메인 화면]
┌─────────────────────────────────┐
│  본인아이디/프로젝트이름          │
│                                 │
│  📁 .streamlit                  │
│  📄 app.py          ← 이거 클릭! │
│  📄 requirements.txt            │
│  📄 README.md                   │
└─────────────────────────────────┘
```

#### 3-2. 편집 모드

```
[app.py 파일 화면]
┌─────────────────────────────────┐
│  app.py                          │
│  ┌───────────────────┐          │
│  │ ✏️ Edit  🗑️ Delete │  ← ✏️ 클릭│
│  └───────────────────┘          │
│                                 │
│  [코드 내용...]                  │
└─────────────────────────────────┘
```

#### 3-3. 내용 교체

1. **코드 전체 선택**
   - Windows: `Ctrl + A`
   - Mac: `Cmd + A`

2. **삭제**
   - `Delete` 키

3. **ESSENTIAL_FILES 폴더에서 app.py 열기**
   - 메모장으로 열기
   - 전체 선택: `Ctrl + A`
   - 복사: `Ctrl + C`

4. **깃허브에 붙여넣기**
   - `Ctrl + V`

#### 3-4. 저장

```
[페이지 맨 아래]
┌─────────────────────────────────┐
│  Commit changes                 │
│  ┌───────────────────┐          │
│  │ Update app.py     │  ← 메시지│
│  └───────────────────┘          │
│  ┌───────────────────┐          │
│  │ [Commit changes]  │  ← 클릭! │
│  └───────────────────┘          │
└─────────────────────────────────┘
```

---

### Step 4: requirements.txt 파일 수정

**위 Step 3과 똑같은 방법으로!**

1. 저장소 메인으로 돌아가기
2. `requirements.txt` 클릭
3. ✏️ 연필 아이콘 클릭
4. 전체 선택 → 삭제
5. ESSENTIAL_FILES의 requirements.txt 내용 복사
6. 붙여넣기
7. "Commit changes" 클릭

---

### Step 5: 배포 확인

#### 5-1. 1-2분 대기

커피 한 모금 ☕

#### 5-2. Streamlit Cloud 확인

1. **Streamlit Cloud 접속**
   - 🔗 [share.streamlit.io](https://share.streamlit.io/)

2. **본인 계정으로 로그인**
   - GitHub 계정으로 로그인

3. **앱 상태 확인**

```
[Streamlit Cloud 대시보드]
┌─────────────────────────────────┐
│  My apps                        │
│  ┌───────────────────┐          │
│  │ 🟢 본인앱이름      │          │
│  │ Running           │          │
│  │ [Open app]        │  ← 클릭! │
│  └───────────────────┘          │
└─────────────────────────────────┘
```

#### 5-3. 앱 접속

새 창에서 앱이 열립니다!

## 🆘 문제 해결

### ❌ "파일을 찾을 수 없습니다"

**해결방법:**

1. 깃허브 저장소에서 **"Add file"** 클릭
2. **"Create new file"** 선택
3. 파일 이름 입력: `app.py`
4. 내용 붙여넣기
5. **"Commit new file"** 클릭

---

### ❌ "수정했는데 배포가 안 됩니다"

**해결방법:**

1. **Streamlit Cloud 접속**
   - 🔗 [share.streamlit.io](https://share.streamlit.io/)

2. **앱 찾기**

3. **우측 ⋮ 메뉴 클릭**

4. **"Reboot app" 선택**

---

### ❌ "오류 메시지가 뜹니다"

**확인사항:**

1. **requirements.txt 정확한지 확인**
   ```
   streamlit==1.29.0
   pandas==2.1.4
   numpy==1.26.2
   plotly==5.18.0
   pyupbit==0.2.33
   pybithumb==1.0.21
   python-dotenv==1.0.0
   ```

2. **app.py 완전히 복사되었는지 확인**
   - 파일 크기: 약 50KB 이상

3. **깃허브에서 파일 다시 열어보기**
   - 코드가 제대로 보이는지 확인

---

### ❌ "파일이 너무 많아요"

**해결방법:**

처음에는 **필수 2개만** 올리세요:
- `app.py`
- `requirements.txt`

작동 확인 후 나머지 추가!
